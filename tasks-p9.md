# P9: Qt 项目可靠生成与可选插件化生态

> 目标：让 OPC 能可靠生成可构建、可验收的 Qt 项目，并建立可选插件机制，使 Qt、后续语言/项目类型能力可按需安装或启用。
> 测试策略：优先跑定向测试，避免全量测试影响本机性能。
> 设计原则：核心 runtime 保持轻量；Qt/C++/其他语言生成能力通过插件或 project pack 扩展；用户不需要的能力默认不加载、不安装、不触发。
